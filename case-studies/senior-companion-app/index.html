    EverWell

    A voice-first wellness companion for seniors

    World's Largest Hackathon presented by Bolt • June 2025

    Inspiration

    We were inspired by the aging journey of our own parents and in-laws, noticing how easily they can slip into isolation or sedentary routines with mobility issues and loss of a partner. As busy parents and working professionals, we continue to struggle to provide caregiving and companionship.

    Caregivers and family members cannot be with them 24/7 and need a way to motivate them to be active and safe. We envisioned EverWell as a kind, voice-first companion that empowers seniors to move more, feel more connected, and age with greater independence — while giving caregivers peace of mind.

    What EverWell Does

    EverWell is a comprehensive wellness companion designed primarily for seniors, with features to support their daily health, activity, and social engagement. It also includes a dedicated portal for caregivers to monitor their loved ones' well-being.

    Core Features

    Smart Weather Detection & Walking Nudges

    Detects weather conditions and provides the perfect time to walk, gently nudging users to stay active.

    Walking Buddy

    Enables light conversation during walks through an AI-powered voice companion, making exercise more enjoyable and less lonely.

    Story Time

    Offers narrated stories across various categories using voice AI assistant for entertainment and cognitive engagement.

    Mood Check-in

    Allows users to log their daily mood, helping track emotional well-being over time.

    Activity Tracking

    Monitors daily steps, distance, calories burned, and active minutes, with progress towards set goals.

    Medication Management

    Provides medication reminders, tracks adherence, and allows users to manage their medication schedules.

    Calendar and Events

    Helps users organize appointments, personal events, and walking activities.

    Safety Features

    Includes emergency contacts management, location sharing with family, and health monitoring capabilities.

    Caregiver Portal

    Offers a separate login for caregivers to view a summary of the senior's daily activities, medication compliance, mood trends, and receive alerts for real-time updates and peace of mind.

    Design Approach

    EverWell features a clean, modern UI with calming gradients, bold typography, and large, accessible components designed specifically for seniors. Visual cues like emojis, icons, and color-coded statuses make navigation intuitive and engaging. The interface feels warm, uncluttered, and emotionally supportive.

    How We Built It

    We leveraged cutting-edge technology to create a robust, accessible mobile application:

    React Native Expo Supabase PostgreSQL ElevenLabs API Netlify
    • Frontend Framework: React Native with Expo (Managed Workflow)
    • Navigation: Expo Router for tab-based and stack navigation
    • Database: PostgreSQL managed via Supabase for data storage, authentication, and real-time capabilities
    • Backend: Supabase Edge Functions and Supabase Auth
    • Local Storage: @react-native-async-storage/async-storage
    • Audio & Speech: expo-av for audio playback, expo-speech for system text-to-speech, and ElevenLabs API for advanced AI voice synthesis
    • Sensors: expo-sensors (Pedometer for step tracking) and expo-location for location services
    • Deployment: Netlify

    Challenges We Faced

    Building a full-featured mobile app in a hackathon setting came with significant challenges:

    • Framework Compatibility: React Native and Expo framework version incompatibility required multiple attempts to resolve, depleting tokens
    • Debugging: Terminal errors required creative debugging approaches, including using screenshots
    • Last-Minute Changes: UI changes at the last minute occasionally broke the app, requiring rapid fixes
    • Version Control: GitHub integration presented learning curves for team coordination

    Accomplishments

    We're proud of what we achieved:

    • Designed a brand identity and icon system that feels calming, clear, and trustworthy
    • Successfully integrated voice AI for interactive companionship features
    • Created a comprehensive caregiver portal for real-time monitoring
